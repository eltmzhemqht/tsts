## ⚠️ 개선이 필요한 부분

### 1. 네이티브 브라우저 다이얼로그 사용

**문제점**:
- `alert()`, `confirm()`, `prompt()` 사용
- 모바일에서 불편함
- 디자인과 일치하지 않음
- 접근성 문제

**위치**:
- `clearRankings()`: `confirm()`, `prompt()`, `alert()` 사용
- `submitRanking()`: `alert()` 사용 (에러 시)

**개선 제안**:
- 커스텀 다이얼로그 컴포넌트 사용 (이미 `Dialog` 컴포넌트 있음)
- Toast 알림으로 변경 (sonner 라이브러리 사용 가능)

### 2. 에러 피드백 부족

**문제점**:
- 네트워크 에러 시 `alert()`만 사용
- 사용자가 에러를 쉽게 놓칠 수 있음
- 재시도 옵션 없음

**위치**:
- `submitRanking()`: 에러 시 `alert()`만 표시
- `fetchRankings()`: 에러 시 콘솔에만 로그

**개선 제안**:
- Toast 알림으로 에러 표시
- 재시도 버튼 제공
- 네트워크 상태 표시

### 3. 게임 종료 시 즉시 반응

**현재 상태**:
- `timeLeft === 0`일 때 `useEffect`에서 처리
- `isGameEndedRef`로 중복 호출 방지
- ✅ 적절히 구현됨

**확인 필요**:
- 게임 종료 시 즉시 결과 화면으로 전환되는지
- 매수/매도 버튼이 즉시 비활성화되는지

### 4. 랭킹 제출 시 피드백

**현재 상태**:
- ✅ "등록 중..." 표시
- ✅ 성공 시 메시지 표시
- ⚠️ 에러 시 `alert()` 사용

**개선 제안**:
- Toast 알림으로 변경
- 더 명확한 에러 메시지

### 5. 입력 검증 피드백

**현재 상태**:
- ✅ `maxLength={10}` 설정
- ✅ `disabled={!playerName.trim() || isSubmitting}` 처리
- ⚠️ 사용자가 왜 버튼이 비활성화되었는지 명확하지 않을 수 있음

**개선 제안**:
- 입력 필드 아래에 힌트 텍스트 표시
- "이름을 입력하세요" 또는 "최대 10자까지 입력 가능"

## 🎯 우선순위별 개선 사항

### 높은 우선순위 (즉시 개선 권장)

1. **alert/confirm/prompt를 커스텀 다이얼로그로 변경**
   - 사용자 경험 크게 개선
   - 모바일 친화적
   - 디자인 일관성

2. **에러 피드백을 Toast로 변경**
   - 더 나은 사용자 경험
   - 자동으로 사라짐
   - 덜 방해적

### 중간 우선순위

3. **입력 검증 피드백 개선**
   - 힌트 텍스트 추가
   - 실시간 검증 메시지

4. **네트워크 상태 표시**
   - 오프라인 감지
   - 재연결 시도 안내

### 낮은 우선순위 (선택적)

5. **애니메이션 최적화**
   - 현재는 적절하지만, 저사양 기기 고려

6. **접근성 개선**
   - 키보드 네비게이션
   - 스크린 리더 지원

## 🔧 구체적 개선 코드 예시

### 1. Toast 알림으로 변경

```typescript
import { toast } from "sonner";

// 기존: alert("랭킹이 초기화되었습니다.");
// 개선:
toast.success("랭킹이 초기화되었습니다.");

// 기존: alert(`랭킹 등록 실패: ${errorMessage}`);
// 개선:
toast.error(`랭킹 등록 실패: ${errorMessage}`);
```

### 2. 커스텀 확인 다이얼로그

```typescript
// 기존: if (!confirm("정말로 모든 랭킹을 초기화하시겠습니까?")) return;
// 개선: Dialog 컴포넌트 사용
```

### 3. 입력 힌트 추가

```typescript
<Input
  type="text"
  placeholder="학번 이름"
  value={playerName}
  onChange={(e) => setPlayerName(e.target.value.slice(0, 10))}
  maxLength={10}
  className="flex-1 bg-slate-900 border-slate-700"
/>
<p className="text-xs text-slate-500 mt-1">
  {playerName.length}/10자 {!playerName.trim() && "(이름을 입력하세요)"}
</p>
```
