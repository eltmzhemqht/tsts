# 2분 투자 챌린지 게임 설명서

**작성일**: 2025-12-25  
**목적**: 게임의 기능, 규칙, 플레이 방법을 상세히 설명

---

## 🎮 게임 개요

### 게임 이름
**2분 투자 챌린지**

### 게임 목표
2분 동안 투자를 통해 초기 자본을 최대한 늘려 랭킹 1위를 달성하는 것

### 게임 시간
**120초 (2분)**

### 초기 자본
**₩20,000,000 (2천만원)**

---

## 💰 투자 가능한 자산

게임 시작 전 3가지 자산 중 하나를 선택해야 합니다.

### 1. 암호화폐 (Coin)
- **아이콘**: 비트코인
- **변동성**: ±10~30%
- **특징**: 고위험, 고수익
- **추천 대상**: 공격적인 투자 전략을 선호하는 플레이어

### 2. 주식 (Stock)
- **아이콘**: 주가 차트
- **변동성**: ±5~15%
- **특징**: 균형 잡힌 투자
- **추천 대상**: 중간 위험을 선호하는 플레이어

### 3. 부동산 (Real Estate)
- **아이콘**: 건물
- **변동성**: ±2~6%
- **특징**: 안정적인 자산
- **추천 대상**: 안정적인 투자를 선호하는 플레이어

## 📰 뉴스 이벤트 시스템

### 뉴스 발생 주기
- **5~8초 간격**으로 랜덤하게 발생
- 게임 시작 시 즉시 첫 뉴스 발생
- 게임 종료 5초 전까지 발생

### 뉴스 종류

#### 좋은 뉴스 (Good News)
- 가격이 **10~25% 상승**
- 예시:
  - "미국 연준, 기준금리 동결 결정" (+15%)
  - "테슬라, 신기술 개발 성공 발표" (+25%)
  - "워렌 버핏, 해당 자산 대량 매수" (+24%)

#### 나쁜 뉴스 (Bad News)
- 가격이 **11~25% 하락**
- 예시:
  - "미국 연준, 기준금리 0.5%p 인상 발표" (-12%)
  - "테슬라, 리콜 발표로 주가 하락" (-24%)
  - "워렌 버핏, 해당 자산 대량 매도" (-25%)

### 가격 변동 타이밍
- 뉴스 발생 후 **2.5초 후** 가격이 변동
- 플레이어가 뉴스를 읽고 반응할 시간 제공

## 🎯 게임 플레이 방법

### 1. 게임 시작
1. 메인 화면에서 **"게임 시작"** 또는 **"튜토리얼 보기"** 버튼 클릭
2. 자산 선택 화면에서 투자할 자산 선택 (암호화폐, 주식, 부동산)
3. 게임 시작

### 2. 게임 플레이
- **매수 (BUY)**: 보유 현금으로 자산을 전액 매수
  - 버튼 클릭 시 보유 현금으로 살 수 있는 최대 수량 자동 구매
  - 가격이 현재 가격보다 낮을 때 매수하면 수익 가능
  
- **매도 (SELL)**: 보유 자산을 전액 매도
  - 버튼 클릭 시 보유한 모든 자산을 현재 가격에 매도
  - 가격이 매수 가격보다 높을 때 매도하면 수익 실현

### 3. 게임 화면 정보
- **남은 시간**: 게임 종료까지 남은 시간 (초 단위)
- **현재 가격**: 선택한 자산의 현재 가격
- **보유 현금**: 매수/매도에 사용 가능한 현금
- **보유 자산**: 현재 보유한 자산 수량
- **총 자산 가치**: (보유 현금 + 보유 자산 × 현재 가격)
- **수익률**: 초기 자본 대비 수익률 (%)
- **가격 차트**: 시간에 따른 가격 변동 그래프
- **뉴스 히스토리**: 발생한 뉴스 이벤트 목록

### 4. 게임 종료
- 시간이 0초가 되면 자동으로 게임 종료
- 최종 자산 가치가 계산되어 결과 화면으로 이동

## 🏆 랭킹 시스템

### 랭킹 등록
1. 게임 종료 후 결과 화면에서 이름 입력 (최대 10자)
2. **"등록"** 버튼 클릭
3. 랭킹에 등록 완료

### 랭킹 기준
- **수익률** 기준으로 정렬
- 수익률이 높을수록 상위 랭킹

### 랭킹 표시
- **1위**: 🏆 금색 트로피
- **2위**: 🥈 은색 메달
- **3위**: 🥉 동메달
- **4위 이하**: 순위 숫자

### 랭킹 확인
- 메인 화면에서 실시간 랭킹 확인 가능
- 결과 화면에서 **"랭킹 보기"** 버튼으로 전체 랭킹 확인
- 랭킹은 2초마다 자동 업데이트

---

## 📚 튜토리얼

### 튜토리얼 시작
- 메인 화면에서 **"튜토리얼 보기"** 버튼 클릭
- 또는 **"게임 시작"** 버튼 클릭 후 자산 선택 화면에서 튜토리얼 진행

### 튜토리얼 내용
1. **자산 선택**: 투자할 자산을 선택하는 방법
2. **매수 버튼**: 자산을 매수하는 방법
3. **매도 버튼**: 자산을 매도하는 방법
4. **뉴스 이벤트**: 뉴스가 가격에 미치는 영향
5. **가격 차트**: 가격 변동을 그래프로 확인하는 방법

### 튜토리얼 특징
- 튜토리얼 진행 중에는 시간이 멈춤
- 각 단계마다 화살표와 설명으로 안내
- 튜토리얼을 건너뛰고 바로 게임 시작 가능

## 🎨 게임 화면 구성

### 메인 화면
- 게임 제목: "2분 투자 챌린지"
- 게임 시작 버튼
- 튜토리얼 보기 버튼
- 실시간 랭킹 (최대 20개)

### 자산 선택 화면
- 3가지 자산 카드
- 각 자산의 설명과 변동성 정보
- 선택 버튼

### 게임 플레이 화면
- 상단: 남은 시간, 현재 가격, 총 자산 가치, 수익률
- 중앙: 가격 차트, 뉴스 히스토리
- 하단: 매수/매도 버튼, 보유 현금/자산 정보

### 결과 화면
- 최종 자산 가치
- 수익률
- 결과 메시지 (수익률에 따라 다름)
- 랭킹 등록 입력 필드
- 랭킹 보기 버튼
- 다시 도전하기 버튼

## 🎯 게임 전략

### 기본 전략
1. **뉴스를 주의 깊게 읽기**: 뉴스가 가격에 큰 영향을 미침
2. **타이밍이 중요**: 뉴스 발생 후 2.5초 후 가격 변동
3. **손절과 익절**: 수익 실현과 손실 최소화의 균형

### 자산별 전략

#### 암호화폐 (고위험, 고수익)
- 큰 변동성으로 빠른 수익 가능
- 하지만 큰 손실 위험도 높음
- 공격적인 매수/매도 타이밍 중요

#### 주식 (균형)
- 적당한 변동성으로 안정적이면서도 수익 가능
- 초보자에게 추천
- 뉴스를 보고 신중하게 판단

#### 부동산 (안정)
- 낮은 변동성으로 안정적
- 큰 수익은 어렵지만 손실 위험도 낮음
- 장기 투자 전략에 적합

## 📊 게임 통계

### 가격 범위
- **초기 가격**: ₩5,000,000 ~ ₩9,000,000 (랜덤)
- **최소 가격**: ₩1,000,000
- **최대 가격**: ₩20,000,000

### 뉴스 이벤트
- **총 뉴스 수**: 40개 (좋은 뉴스 20개, 나쁜 뉴스 20개)
- **발생 주기**: 5~8초 간격
- **가격 변동 지연**: 2.5초

### 랭킹 제한
- **이름 길이**: 최대 10자
- **수익률 범위**: -100% ~ 10,000%
- **최종 자산 범위**: 0 ~ 100,000,000,000원

## 🎓 게임 팁

1. **뉴스를 먼저 읽기**: 가격 변동 전에 뉴스를 읽고 판단
2. **차트 확인**: 가격 추세를 파악하여 매수/매도 결정
3. **손절 중요**: 큰 손실을 방지하기 위해 적절한 손절
4. **익절 타이밍**: 수익이 나면 적절한 시점에 매도
5. **여러 번 플레이**: 다양한 자산과 전략 시도

## 🏅 결과 메시지

수익률에 따라 다른 메시지가 표시됩니다:

- **+50% 이상**: "투자천재의 탄생! 워렌 버핏이 형님이라 부르겠네요."
- **+20% 이상**: "훌륭한 감각입니다! 야수의 심장을 가지셨군요."
- **+0% 이상**: "은행 이자보다는 낫네요! 소소한 수익 축하합니다."
- **-20% 이상**: "수업료 냈다고 생각하세요... 다음엔 더 잘할 수 있습니다."
- **-20% 미만**: "아쉽네요! 다시 도전해보세요."
